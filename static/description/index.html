Employee Loan Schedule

By Accurate IT Solutions

This Employee Loan Schedule module is designed to efficiently manage employee loans within an organization. It provides a structured approach to handling loan requests, and approvals. The module allows users to define various loan types, specifying details such as loan amount limits, terms, and interest rates. Employees can apply for loans, which go through a customizable approval workflow involving department heads and HR, ensuring proper oversight.

Description with screenshots

The Employee Loan Schedule module is to Manage Employee loans. It contains loan types and manages loan applications.

The User can navigate to the Loan Types menu. Here, they can create new loan types by setting details like loan limit, loan term, interest rate, and whether to apply interest. Loan and interest accounts can also be linked to track financial records.

In Loan Request form.By filling up the details like Employee, Department, Department, Manager, Start Date, Job Position, User, Payment Method, Loan Type, Date, Term, End Date and also add the compute fields like Loan Amount, Interest Amount, Paid Amount, Remaining Amount, Installment Amount, Interest Rate, Interest Type, Installments.

When click on compute Installment the module automatically calculates the number of installments, including the interest amount and total repayment. The user can view each installment's details, including the due date and remaining balance.

All the computed fields that can be viewd as a tree view.

When Click on Submit Request the loan goes through various approval stages.

Department Managers can review the loan and either approve or reject it.

When the Department Manager Approves the loan requests it goes for HR Approval

A button named Loans is added to display all the Loan Requests that Department Manger Approves.

When clicked on Loans Button, it opens a window showing a list of loan records that the Department Manager Approves the loan for a employee. This view includes details like loan reference, loan amount, interest rate, and status. It provides a quick way for HR to review and manage loans for a specific employee.
